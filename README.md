# Cháo Lươn Xứ Nghệ - Landing Page

Dự án Landing Page cho quán "Cháo Lươn Xứ Nghệ" được xây dựng với Next.js, Tailwind CSS và Framer Motion.

## Tính Năng Chính
- **Giao diện hiện đại**: Thiết kế theo phong cách ấm cúng, đậm chất Nghệ An.
- **Tương tác mượt mà**: Hiệu ứng cuộn trang và hover đẹp mắt với Framer Motion.
- **Responsive**: Hiển thị tốt trên mọi thiết bị (Mobile, Tablet, Desktop).
- **Đặt hàng nhanh**: Form đặt hàng đơn giản, tiện lợi.

## Cài Đặt và Chạy Dự Án

Do dự án này được tạo thủ công (Manual Scaffolding), hãy đảm bảo bạn đã cài đặt **Node.js** (phiên bản 18+).

1. **Cài đặt dependencies**:
   Mở terminal tại thư mục dự án và chạy:
   ```bash
   npm install
   ```

2. **Chạy server phát triển (Development Server)**:
   ```bash
   npm run dev
   ```
   Truy cập [http://localhost:3000](http://localhost:3000) để xem trang web.

3. **Build cho Production**:
   ```bash
   npm run build
   npm start
   ```

## Cấu Trúc Thư Mục
- `/app`: Chứa các trang chính (page.tsx, layout.tsx).
- `/components`: Chứa các thành phần giao diện (Header, Hero, Menu, v.v.).
- `/public`: Chứa tài nguyên tĩnh (hình ảnh, fonts).
